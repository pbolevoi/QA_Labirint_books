# QA Automation Demo (Selenium)

Небольшой учебный проект по автоматизированному тестированию веб-приложения.
Проект демонстрирует базовые навыки автоматизации UI тестов на Python с использованием Selenium.

## Технологии

* Python
* Selenium WebDriver
* WebDriverWait / Expected Conditions

## Что тестируется

В тестах используется интернет-магазин книг **labirint.ru**.

Реализованные проверки:

1. Поиск книги через строку поиска
2. Переход на страницу товара
3. Добавление книги в корзину
4. Проверка счётчика корзины
5. Добавление книги в корзину со страницы поиска
6. Проверка отображения кнопки увеличения количества товара

## Структура тестов

Основные действия вынесены в функции:

* открытие главной страницы
* поиск товара
* выбор книги из результатов
* добавление товара в корзину
* проверка состояния корзины

Тесты:

* `test_1` — поиск книги и добавление её в корзину со страницы товара
* `test_2` — добавление книги в корзину со страницы поиска
